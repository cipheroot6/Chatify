Chatify is a modern real-time chat application built with the MERN stack (MongoDB, Express, React, Node.js). Users can communicate through text messages and image sharing in a sleek, dark-themed interface.
Key Features:
- User Authentication: Secure signup and login with JWT, session management via cookies, and protected routes.
- Real-time Messaging: Instant message delivery between users with conversation history stored in MongoDB.
- Image Sharing: Share images in conversations, managed through Cloudinary CDN.
- Profile Management: Customizable profiles with profile pictures and full names.
- Modern UI: Sleek dark theme built with React, Vite, and Tailwind CSS, featuring animated backgrounds and responsive design.
- Security: Arcjet protection, email verification via Resend, and secure password handling.
- Deployment Ready: Configured for production with static file serving.
Tech Stack:
- Frontend: React, Vite, Tailwind CSS, React Router
- Backend: Express.js, MongoDB/Mongoose
- Services: Cloudinary, Resend, Arcjet
